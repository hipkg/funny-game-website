# 🎮 Funny Game Website

A super fun, animated, and interactive web-based game with registration, gameplay, and leaderboard features!

## ✨ Features

### 1. 📝 Registration Page
- 👤 User registration with Name and Email
- 😄 Avatar selection with 8 different funny emoji avatars
- ✅ Terms and Conditions checkbox (checked by default)
- 🎨 Animated form with smooth transitions
- 🖱️ Hover effects and interactive animations

### 2. 🎮 Game Page
- 🎯 **"Catch the Numbers!"** - Click on the correct number within 30 seconds
- ⭐ Score tracking with real-time updates
- ⏱️ Animated timer countdown
- 🎉 Celebration effects when correct numbers are clicked
- 🌈 Smooth animations and transitions
- 📊 Player info display with avatar

### 3. 🏆 Leaderboard
- 🥇 Top 10 players ranked by score
- 🎖️ Medals for top 3 players (Gold, Silver, Bronze)
- ✨ Persistent storage using localStorage
- 🔄 Play again functionality
- 🎊 Celebration animations

## 🎨 Design Highlights

- 🌊 **Animated Gradient Background** - Smooth color transitions
- ✨ **Floating Animations** - Elements gently float up and down
- 🎯 **Interactive Elements** - Buttons and cards respond to hover/click
- 🎉 **Emoji Rain** - Celebration effects during gameplay
- 📱 **Fully Responsive** - Works great on mobile and desktop
- 🎪 **Fun Emojis** - Throughout the interface for a playful feel

## 🎮 How to Play

1. **Register** 📝
   - Enter your funny name
   - Add your email
   - Pick your favorite avatar
   - Agree to have a funny time!

2. **Play** 🎮
   - A number will appear with 8 other random numbers
   - Click on the correct number as many times as possible
   - You have 30 seconds!

3. **Score** ⭐
   - Your score is recorded automatically
   - See how you rank against other players

4. **Leaderboard** 🏆
   - View the top 10 players
   - See your position
   - Play again to improve your score

## 🚀 Getting Started

1. Clone or download the repository
   ```bash
   git clone https://github.com/hipkg/funny-game-website.git
   cd funny-game-website
   ```

2. Open `index.html` in your web browser
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Or just double-click index.html
   ```

3. Start playing! 🎉

## 🎯 Game Rules

- ⏱️ You have exactly 30 seconds to play
- 🎯 A target number will appear
- 🖱️ Click on the correct number among 9 options
- ✨ Each correct click gives you 1 point
- 🌊 Numbers are shuffled each round
- 🏁 Game ends when time runs out
